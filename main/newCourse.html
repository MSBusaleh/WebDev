WebDev course (front end)



Course subject

This course talks about how to build a web page using HTML (Hyper Text Markup Language)
then style it with CSS (Cascading Style Sheets), after that we will use JavaScript.
In this course we will only study the fundamentals of HTML, and later we will learn how to integrate CSS to make our HTML page look more attractive.


Course Goals

  1. Learn what is HTML and what is it used for
  2. Learn the elements that construct the web page in HTML
  3. Build the first web page with HTML together
  4. Learn what is CSS and what is it used for
  5. Learn the important concepts in CSS
  6. Style our page using CSS
  7. Start learning JS


Course levels

This course will be consisted of 4 levels:
1st level: Intro. to HTML
2nd level: Practice and work
3rd level: Intro. to CSS & practice
4th level: intro. to JS
