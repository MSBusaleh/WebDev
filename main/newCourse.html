HTML course



Course subject

This course talks about how to build a web page using HTML (Hyper Text Markup Language). In this course we will only study the fundamentals of HTML, and later we will learn how to integrate CSS to make our HTML page look more attractive.


Course Goals

  1. Learn what is HTML and what is it used for
  2. Learn the elements that construct the web page in HTML
  3. Build the first web page together


Course levels

This course will be consisted of 2 levels:
1st level: Intro. to HTML
2nd level: Practice and work
